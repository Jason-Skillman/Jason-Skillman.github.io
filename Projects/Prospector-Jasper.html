Prospector Jasper

Unity Game
Associate Capstone Project

Project Manager & Lead Unity Programmer
Team Size: 2

Description:
Prospector Jasper is a game inspired by Dig Dug which is a classic arcade game. The game takes place underground, mining your way through each level fighting monsters and collecting gems and ores.

My Contribution:
Lead the overall process for creating this game. I programmed the entire level system which gets procedurally generated. Each level generation goes through many passes to created the level. Some of the passes include horizontal and vertical tunnels, cave lights, gems, monsters and bombs. I worked on refining the earthquake feature which would create more tunnels, spawn more monsters and shake the screen. Game went through lots of user testing, mostly with kids and received great feedback. Also worked on a mobile port of the game for Android.

Challenges:
• My vision for this game had to be carefully explained to my designer so we could get the theme correct.
• Art had to go though many iterations.
• Refined the procedural level generation to created unique levels. No two levels are the same.
• Had to meet dead lines within a ten week work sprint.
